# Sistema de Gestión de Flujo de Fondos

## Contexto
El objetivo es migrar una gestión de flujo de caja actualmente en Excel a una aplicación web robusta para evitar errores de carga y pérdida de integridad de datos.

## Stack Tecnológico
- **Framework**: Laravel 11
- **Frontend**: Livewire 3
- **UI Components**: MaryUI (Tailwind CSS + DaisyUI)
- **Base de Datos**: MySQL / PostgreSQL

## Objetivos Principales
1. Registrar movimientos diarios (Ingresos/Egresos).
2. Controlar saldos de Caja y Banco por separado.
3. Clasificar gastos en Fijos y Variables.
4. Generar reportes comparativos mensuales y anuales.
